# 📊 Analítica Integral de Clientes: Predicción de Abandono y Segmentación

## 📌 Descripción del proyecto

Este proyecto desarrolla un enfoque integral de analítica de clientes aplicado al ámbito del marketing y la gestión de la relación con el cliente.

A lo largo de dos fases complementarias, se aplican técnicas de machine learning supervisado y no supervisado con el objetivo de comprender el comportamiento del cliente y diseñar estrategias de segmentación basadas en datos.

El proyecto integra clasificación y clustering dentro de un mismo marco analítico, simulando un entorno real de toma de decisiones en negocio.

---

## 🎯 Objetivos

- Desarrollar un pipeline completo para la predicción del abandono.
- Aplicar técnicas de segmentación no supervisada para identificar grupos homogéneos de clientes.
- Integrar riesgo, rentabilidad y segmentación en un mismo enfoque estratégico.
- Demostrar la aplicación práctica de modelos de machine learning dentro de un contexto realista de negocio.

---

## 🗂️ Estructura del proyecto

### 1️⃣ Predicción de abandono (Clasificación)

- Análisis exploratorio del comportamiento del cliente.
- Preparación y transformación de variables.
- Desarrollo de un pipeline de modelado.
- Evaluación del modelo con métricas adecuadas al problema de negocio.
- Combinación de distintos modelos para determinar cuál es más efectivo.

Objetivo: anticipar la pérdida de clientes y facilitar decisiones de retención basadas en datos.

---

### 2️⃣ Segmentación de clientes (Clustering)

- Preparación y escalado de variables.
- Aplicación de técnicas de clustering (aprendizaje no supervisado).
- Selección del número óptimo de clusters.
- Interpretación de los segmentos obtenidos.
- Traducción de los resultados a posibles estrategias comerciales diferenciadas.

Objetivo: identificar grupos de clientes con características similares para diseñar acciones de marketing personalizadas.

---

## 🛠️ Tecnologías utilizadas

- Python  
- Pandas  
- Scikit-learn  
- Matplotlib  
- Jupyter Notebook  

---

## 📁 Datos

El dataset simula un entorno de analítica de clientes, incluyendo variables demográficas, contractuales y de comportamiento.

Se ha realizado un proceso previo de limpieza, tratamiento de valores faltantes, transformación de variables y preparación para modelado.

---

## 🚀 Enfoque del proyecto

El objetivo principal no es construir el modelo más complejo posible, sino demostrar la capacidad de:

- Definir correctamente un problema de negocio.
- Seleccionar técnicas adecuadas según el objetivo.
- Evaluar modelos con criterio.
- Traducir resultados técnicos en implicaciones estratégicas.
